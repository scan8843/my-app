# Invoice App

청구서 작성 및 문서(DOCX / PDF) 생성 웹 애플리케이션

---

## 📌 프로젝트 소개

이 프로젝트는 사용자가 품목을 입력하고,
자동으로 청구서를 생성하여 Word(DOCX) 또는 PDF로 다운로드할 수 있는 웹 앱입니다.

---

## 🚀 주요 기능

- ✅ 품목(물품) 입력 및 관리
- ✅ 수량 / 단가 기반 자동 금액 계산
- ✅ 전자 서명 입력 (캔버스)
- ✅ Word(DOCX) 문서 생성 및 다운로드
- ✅ PDF 변환 기능 (추가 예정)

---
